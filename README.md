# 🎬 Movie Explorer Web App  

A React-based movie browsing platform powered by TMDB API.  

This project is a fully functional movie web application where users can explore movies, view detailed information, and check actor profiles in a clean and interactive UI.  

---

## 📌 Project Objectives  

- Build a responsive movie browsing application using React  
- Integrate real-time movie data using TMDB API  
- Implement routing for multiple pages (Home, Popular, Details)  
- Display detailed movie and cast information  
- Enable navigation between movies and actors  
- Create a clean and user-friendly UI  

---

## ✨ Features  

### 🎥 Movie Browsing  
- View popular and romantic movies together on the Home page  
- Separate Popular Movies page  
- Clean grid layout  

### 🔄 Pagination  
- Next / Previous navigation on Home page  
- Loads new movies dynamically  

### 🎬 Movie Details  
- Poster, Title, Overview  
- Rating ⭐ and Release Date 📅  
- Top Cast section  

### 👨‍🎤 Actor Details  
- Click on cast members to view details  
- Shows Name, Birthday, Popularity, Biography  
- Handles missing data and images gracefully  

### 🎨 UI Enhancements  
- Responsive design  
- Image fallback (no broken images)  
- Smooth hover effects  
- Footer branding: *By Ankit Chaudhary*  

---

## 🚀 Key Highlights  

- Uses real-time data from TMDB API  
- Handles missing API data properly  
- Component-based architecture  
- Dynamic routing using React Router  
- State management using React Hooks and Redux

2. Navigate to project folder
